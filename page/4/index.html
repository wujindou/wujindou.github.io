Hadoop2.2安装使用

By Jackie wu

发表于 2015-04-17

配置ssh

  • 生成ssh-key 在

    ssh-keygen -t rsa -P ""
cd ~/.ssh
cat id_rsa.pub>> authorized_keys

    Jdk环境配置

    Linux开发环境，可以删除系统原来的jdk,查找下载安装最新版本的jdk

    • rpm -qa |grep jdk ,rpm -qa | grep gcj,
    • 删除执行：rpm -e —nodeps xxx
    • 下载最新版本的JDK.gz解压后放到/usr/local/java目录下

    • 修改配置/etc/profile文件

      JAVA_HOME=/usr/local/java/jdk1.8.0_05
CLASSPATH=.:$JAVA_HOME/lib/tools.jar
 PATH=$PATH:$JAVA_HOME/bin
export JAVA_HOME CLASSPATH PATH
    • 配置生效后执行java -version 看是否和安装的java版本匹配

      Hadoop环境配置

    • 配置~/.bashrc文件
export JAVA_HOME=/usr/local/java/jdk1.8.0_05
export HADOOP_INSTALL=~/hadoop/hadoop-2.2.0
export HADOOP_MAPRED_HOME=$HADOOP_INSTALL
export HADOOP_COMMON_HOME=$HADOOP_INSTALL
export HADOOP_HDFS_HOME=$HADOOP_INSTALL
export YARN_HOME=$HADOOP_INSTALL
export HADOOP_COMMON_LIB_NATIVE_DIR=$HADOOP_INSTALL/lib/native
export PATH=$PATH:$HADOOP_INSTALL/sbin
export PATH=$PATH:$HADOOP_INSTALL/bin
  • 配置core-site.xml 
<configuration>
   <property>
    <name>fs.default.name</name>
    <value>hdfs://127.0.0.1:9000</value>
    <final>true</final>
    </property>
    <property>
    <name>hadoop.tmp.dir</name>
    <value>/home/hadoop/hadoop_tmp</value>
  </property>
</configuration>
  • 配置mapred.xml文件
<configuration>
<property>
        <name>mapreduce.framework.name</name>
        <value>yarn</value>
</property>
</configuration>
  • 配置hdfs-site.xml文件
<configuration>
<property>
       <property>
    <name>dfs.replication</name>
    <value>1</value>
</property>
</configuration>
  • 配置Yarn-site.xml文件（默认就行了）
  • 启动
bin/hdfs namenode -format
显示 ：SHUTDOWN_MSG: Shutting down NameNode at xxx表示成功
启动namenode:
sbin/hadoop-daemon.sh start namenode
sbin/hadoop-daemon.sh start datanode
启动Manage管理
sbin/yarn-daemon.sh start resourcemanager
sbin/yarn-daemon.sh start nodemanager
测试：jps 通过：ip:50070查看
测试hdfs:
mkdir input
cd input
echo "hello world" > test1.txt
echo "hello hadoop" > test2.txt
cd ..
把input目录中的文件拷贝到dnfs上，并保存在in目录中
bin/hadoop dfs -put input/ /in
bin/hadoop dfs -ls /in

Jquery简单封装插件

By Jackie wu

发表于 2015-04-17

由于开发过程是需要使用fancybox，但是已有的结构需要每次都把html代码如下copy到需要添加文档，然后需要每次复制函数。重复早轮子，
于是考虑通过jquery进行简单的封装以避免重复的造轮子工作。
新建fancybox-widget.js 

$(function(){
        var tbox =$(
        '<div id="show-message" style = "display:none"><div class = "cb50" ></div><p class = "t_center ml_25 mr_25 f14 f_red"></p><div class = "cb20" ></div><a href = "javascript:$.fancybox.close();" class = "f14 f_white fc t_center" style = "width:100px; height:30px; line-height:30px; display:block; background:#eb2f7b; border-radius:4px;" > 确认 </a> </div ></div>');
    $('body').prepend(tbox);

    });
    jQuery.showbox = function(data,that ){

    if (data && data.code == 1) {
    $("#show-message p").removeClass('f_red');
    $("#show-message p").addClass('f_green');
    } else {
    $("#show-message p").removeClass('f_green');
    $("#show-message p").addClass('f_red');
    }
    $("#show-message p").html(data.message);
    if (data && data.code == 1) {
    $.fancybox.open({
    'href': "#show-message",
    'scrolling': 'no',
    'modal': true,
    'closeBtn': true,
    'afterClose': function() {
    window.location.reload();
    return;
    },
    'autoSize': false,
    'fitToView': false,
    'width': 300,
    'height': 200,
    'padding': 0
    });

    } else {
    $.fancybox.open({
    'href': "#show-message",
    'scrolling': 'no',
    'modal': true,
    'closeBtn': true,
    'afterClose': function() {
    if (that) {
    $(that).attr("data-tag", '0');
    }
    return;
    },
    'autoSize': false,
    'fitToView': false,
    'width': 300,
    'height': 200,
    'padding': 0
    });
    }
    }
    在需要使用该插件的页面通过引入该js文件，然后调用也非常的简单： 
    $('#show').click(function() {
    var data = { code: 1, message: 'hello' };
    $.showbox(data);

    });
