By Jackie wu

發表於 2015-04-17

Github使用问题

好久没有使用github了，今天决定把blog的东西整理一下，以前写的一些文章也决定放到blog上去，另外发现blog的主题有点太low了，顺便把主题给换下，然后整理文章放上去。
直接图示吧：

  1. 生成public key ,然后添加到ssh key中
  2. 测试

    问题出来了：网上找到了参考

  3. 建立config文件添加以下内容：

    Host github.com
    User git
    Hostname ssh.github.com
    PreferredAuthentications publickey
    IdentityFile C:\Users\wujindou.ssh\id_rsa
    Port 443

  4. 查看config设置,简单的设置下,user.name,user.email


  5. 再项目中添加deploy key授权发布权限

done！！！！
